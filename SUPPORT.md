# Support

Harnery is a solo, pre-1.0 project, so help is best-effort.

- **Bug or unexpected behavior:** open an issue with a minimal reproduction (command, expected, actual).
- **Feature idea:** open an issue describing the use case before the implementation.
- **Security report:** follow [SECURITY.md](SECURITY.md). Do not open a public issue for a vulnerability.

For usage questions, the docs at https://harnery.com are the first stop.
