## What's inside

<CardGrid stagger>
  <Card title="Coord layer" icon="puzzle">
    `.harnery/active/` heartbeats, claim-time guards, commit-time guards, the canonical event stream, harness adapters for Claude Code / Cursor / Codex.
  </Card>
  <Card title="Portable utilities" icon="laptop">
    `tokens`, `eml`, `env`, `grep`, `docs`, `browse`, `fetch`, `read`, `devtools`. Cross-platform and dependency-light, with sensible defaults out of the box.
  </Card>
  <Card title="Standalone web UI" icon="open-book">
    `harn web up` boots a local Next.js dashboard for coord state, councils, workflows, isolated workspaces, and per-project introspection.
  </Card>
  <Card title="Backup + sync" icon="rocket">
    `harn backup` snapshots `.harnery/` via [restic](https://restic.net/); `harn sync` keeps a curated subset live across machines via rclone (Google Drive or any remote).
  </Card>
</CardGrid>
