## Coord layer

- `harn agents …`: heartbeats, peers, claims, task focus, councils
- `harn scratch …`: per-agent markdown journal
- [`harn decision`](./decision/): decision docket — file/resolve decisions instead of blocking on a human

## Files + grep

- `harn grep`: monorepo-aware code search with sensible default excludes
- `harn outline`: TS/JS AST outline + markdown TOC
- `harn callers`: find references to a symbol

## Web ops

- `harn browse`: Playwright-driven web automation
- `harn fetch`: cookie-aware HTTP with shared jar
- `harn read`: HTML → markdown via Readability + Turndown

## Utilities

- `harn tokens`: offline token counting (o200k_base)
- `harn eml`: Gmail .eml → clean chronological markdown
- `harn env`: runtime + tooling environment report
- [`harn docs`](./docs/): markdown freshness, frontmatter, lint, sweep, and index

## Lifecycle

- `harn web up`: standalone Next.js dashboard
- `harn backup`: restic-backed snapshots
- `harn sync`: Google Drive-backed cross-machine sync
- `harn doctor`: runtime + dependency check
- `harn tunnel`: IP-gated Cloudflare quick tunnel(s); run several at once with `--name`
