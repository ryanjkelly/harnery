Every command below links to its own page. Run `harn <command> --help` for the live flag list.

## Coord layer

- [`harn agents`](./agents/): heartbeats, peers, claims, task focus, councils
- [`harn scratch`](./scratch/): per-agent markdown journal that survives compaction
- [`harn decision`](./decision/): decision docket â€” file/resolve decisions instead of blocking on a human
- [`harn session`](./session/): wrap a command to capture it to the coordination ledger
- [`harn context`](./context/): one-screen orientation snapshot (self, repo, peers)
- [`harn presence`](./presence/): read/set mobile vs office presence

## Harnesses + workflows

- [`harn harness`](./harness/): registered Claude/Codex/Cursor capabilities and offline conformance bench
- [`harn policy`](./policy/): validate host-enforced workflow policy documents
- [`harn workflow`](./workflow/): bounded, schema-gated multi-harness workflow scripts

## Files + code navigation

- [`harn grep`](./grep/): monorepo-aware code search with sensible default excludes
- [`harn outline`](./outline/): code outline via the TS AST (PHP/Python by regex)
- [`harn callers`](./callers/): find references to a symbol, classified by kind
- [`harn toc`](./toc/): markdown header outline
- [`harn section`](./section/): extract one markdown section by header
- [`harn file-history`](./file-history/): concise per-file git log with line impact
- [`harn config-get`](./config-get/): read one value from a JSON/YAML config by dotted path
- [`harn edit-batch`](./edit-batch/): one find/replace across many files in a single call

## Web ops

- [`harn fetch`](./fetch/): cookie-aware HTTP with a shared jar

## Lifecycle

- [`harn init`](./init/): wire harnery into a project

- [`harn backup`](./backup/): restic-backed snapshots
- [`harn sync`](./sync/): Google Drive-backed cross-machine sync
- [`harn doctor`](./doctor/): runtime + dependency check
- [`harn tunnel`](./tunnel/): IP-gated Cloudflare quick tunnel(s); run several at once with `--name`
- [`harn completion`](./completion/): shell tab completion (bash/zsh/fish)
